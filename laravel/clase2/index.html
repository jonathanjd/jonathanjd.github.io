Instalación

Instalación

Pasos:

  1. Crear una carpeta en mis documentos con el nombre de proyectos
  2. Abrir la consola de window(CMD)
  3. Colocarnos en la raiz de la carpeta proyectos
  4. Luego: composer create-project --prefer-dist laravel/laravel NombreDelProyecto "5.3.*"
  5. Nombre del Proyecto myLaravel

Paquetes

Instalación:

  1. Paquete Carbon: composer require nesbot/carbon
  2. Paquete DebugBar: composer require barryvdh/laravel-debugbar:~2.4
  3. Getting Started Carbon
  4. Getting Started DebubBar
